# mvi-policy-poc

MVI GitHub 운영 정책 PoC 저장소입니다.

## 운영 원칙

- GitHub Organization을 소스 관리와 Issue Tracking의 기준점으로 사용합니다.
- 1차 범위는 Free 티어에서 가능한 항목으로 제한합니다.
- 지사 간 VPN, self-hosted runner, Power Automate Premium, Enterprise SSO는 이번 범위에서 제외합니다.
- Free 티어 private repository의 강제 통제 한계는 문서와 템플릿 기반 운영 규칙으로 보완합니다.

## 관련 문서

- `docs/free-tier-governance.md`: Free 티어 적용 범위와 보류 항목
- `CONTRIBUTING.md`: 브랜치/PR 운영 규칙
- `.github/ISSUE_TEMPLATE/`: 운영 Issue 양식
